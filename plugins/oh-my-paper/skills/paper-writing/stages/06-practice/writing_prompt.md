# 教学实践应用 写作指南

## 1. 核心任务

本章（第4章）需呈现程序在真实教学场景中的应用，需包含：
- 具体教学案例的设计与实施
- 教学效果的实证评价
- 师生使用反馈
- 虚拟实验与真实教学的融合策略

## 2. 内容要素

### 2.1 教学案例设计

选取 1-2 个典型教学案例，详细描述：

1. **案例背景**
   - 课程名称、年级、学生层次
   - 所在章节与教学目标

2. **教学过程**
   - 课前准备（程序环境预置、学生分组）
   - 课中环节（引入→探究→验证→总结的时间分配）
   - 程序在哪个环节、以何种方式介入

3. **学生活动设计**
   - 学生使用程序的具体操作（如：设计婚配组合→观察后代概率→填写实验报告）
   - 学生应形成的认识（概念性知识 vs 程序性知识）

### 2.2 教学效果评价

1. **评价方式**（可多选）
   - 学生访谈/问卷调查（满意度、有用性、易用性）
   - 学生成绩对比（前测/后测）
   - 课堂观察记录（学生参与度、提问质量）
   - 学生作品评分（系谱图绘制的正确性）

2. **数据呈现**
   - 使用表格呈现问卷调查结果（维度：有用性、易用性、参与度、总体满意度）
   - 使用图表呈现前后测成绩对比
   - 引用典型学生反馈原话

### 2.3 典型使用场景

描述 2-3 个程序使用的典型场景，体现不同教学目标下的差异化应用：
- 场景1：探究性学习（学生自主探索遗传规律）
- 场景2：验证性学习（学生验证教师/教材提出的假设）
- 场景3：复习课（学生用程序梳理遗传方式特征）

### 2.4 教学反思

- 程序使用的优势（相比传统讲授、相比真实实验）
- 程序使用的局限（如：无法培养真实实验操作技能）
- 改进建议（功能增强、配套资源开发）

## 3. 结构规范

建议结构：
```
4 教学实践应用
  4.1 教学案例设计
    4.1.1 案例一：[课名]（约400字）
    4.1.2 案例二：[课名]（约400字）
  4.2 教学效果评价
    4.2.1 评价方法
    4.2.2 结果与分析
  4.3 教学反思
```

## 4. 常见问题

| 问题 | 解决 |
|------|------|
| 教学案例写成程序功能介绍 | 以"学生要完成什么学习任务"为主语，而非"程序有什么功能" |
| 效果评价缺乏数据 | 至少引用3-5条学生原话作为定性证据 |
| 与实验教学的关系不清晰 | 明确说明虚拟实验"替代"什么、"补充"什么 |
