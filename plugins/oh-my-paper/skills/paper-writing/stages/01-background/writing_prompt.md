# 背景介绍 写作指南

## 1. 核心任务

本章（问题提出）需建立"问题→现状→意义→目标"的逻辑链，吸引读者（审稿人/教师）继续阅读。

## 2. 内容要素

### 2.1 问题背景

从一线教学的真实痛点切入（通常1-2段）：
- 高中生物遗传规律教学的核心困难是什么
- 学生常见的理解误区或记忆困境（如：机械记忆"无中生有为隐性"口诀）
- 真实遗传实验的开展障碍（果蝇培养周期、社区调研的隐私问题）

### 2.2 国内外研究现状

文献综述（2-3段），不宜罗列文献，需按主题分类：
- 传统遗传教学策略（模型教具、黑板绘图）的优缺点
- 虚拟实验/仿真程序在理科教学中的应用（引用2-3篇文献）
- AI 辅助开发教学资源的新趋势（引用本研究组的前期工作）
- 现有虚拟遗传实验工具的不足（功能局限、交互性差、无AI辅助）

### 2.3 研究意义

明确本文的增量贡献（不宜夸大）：
- 为遗传规律教学提供交互式探究工具
- 为教师提供 AI 辅助开发虚拟实验的完整范本
- 程序本身可作为开源资源共享

### 2.4 研究目标

简明列出本文要完成的具体目标（通常3点）：
1. 开发支持5种遗传模式的系谱图模拟程序
2. 通过迭代优化提升程序功能完整性
3. 开展教学实践验证程序的教学效果

## 3. 结构规范

建议结构：
```
1 问题的提出
  1.1 遗传规律教学中的困难
  1.2 虚拟实验的研究现状
  1.3 研究意义与目标
```

## 4. 常见问题

| 问题 | 解决 |
|------|------|
| 问题背景写成教材内容 | 从学生真实困难出发，不是知识罗列 |
| 文献综述变成流水账 | 按主题/立场组织，每段有小结句 |
| 研究意义过于泛化 | 具体到"本文"能做什么，不是一般性意义 |

## 5. 参考模板

参考期刊模板.md中"1 问题的提出"部分的格式要求进行写作。引言部分通常 800-1200 字。
