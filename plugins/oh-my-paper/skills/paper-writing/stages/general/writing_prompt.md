# 通用写作/修订指南

## 1. 核心任务

本指南用于论文的**整体修订**和**定稿前检查**，确保全稿逻辑连贯、格式统一、术语一致。

## 2. 检查清单

### 2.1 逻辑连贯性

- [ ] 摘要的四要素（目的、方法、结果、结论）是否完整且呼应正文
- [ ] 引言的"背景→现状→空白→目标"逻辑链是否清晰
- [ ] 每章开头是否有承上启下的过渡句
- [ ] 结论是否与引言提出的目标一一对应

### 2.2 术语一致性

- [ ] 全稿统一使用同一术语（如："系谱图"而非混用"家族图谱"）
- [ ] 遗传学专业术语使用正确（如：基因型、表现型、携带者、显性、隐性）
- [ ] 缩写在首次出现时已定义（如：AI、Google AI Studio）
- [ ] 程序功能名称全书统一（如："VERIFY 模式"而非混用"验证模式"）

### 2.3 格式规范

- [ ] 各级标题编号一致（1 / 1.1 / 1.1.1）
- [ ] 表格有标题（在上方）、图有图注（在下方）
- [ ] 参考文献格式统一（符合《生物学教学》期刊要求）
- [ ] 单位使用标准格式（如："代"而非"代"）

### 2.4 图表质量

- [ ] 表格无竖线（符合期刊习惯），行距清晰
- [ ] 图片分辨率足够（≥300 dpi 或矢量图）
- [ ] 图注能独立说明（不看正文也能理解）
- [ ] 表格数据有实际数值（非"[待补充]"）

### 2.5 引用检查

- [ ] 正文引用序号与参考文献一一对应
- [ ] 引用格式符合期刊要求（顺序编码制）
- [ ] [1] 类期刊文献需含 DOI
- [ ] 所有 \cite{} 引用在 reference.bib 中有对应条目

### 2.6 摘要检查

- [ ] 摘要字数：150-250 词
- [ ] 不使用"本文"、"本研究"等第一人称表述
- [ ] 不出现缩写（除非是领域通用缩写如：AI、DNA）
- [ ] 关键词 3-5 个，与文章内容高度相关

## 3. 常见问题

| 问题 | 解决 |
|------|------|
| 全稿风格不一致 | 统一为正式书面语，避免口语化 |
| 表格格式混乱 | 参照期刊模板中已有表格的格式 |
| 图表序号不连续 | 全文图表统一编号（表1、表2…；图1、图2…） |
| 参考文献缺项 | 按期刊要求补全：作者. 题名[J]. 刊名, 年, 卷(期): 页码. |

## 4. 输出要求

完成修订后，输出：
1. **修订清单**（勾选上述检查项）
2. **发现的待修改项**（列出具体位置，如"第3章第2段术语不一致"）
3. **修订后的文稿**（标注修改位置和修改原因）
