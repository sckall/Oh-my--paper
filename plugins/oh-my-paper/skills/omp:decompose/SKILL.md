# omp:decompose - Problem Decomposition

Use this skill to decompose a research problem.

## Invocation

```
/omp:decompose
```

## Stage

A2 - 问题分解

## Tasks

1. Identify core research question
2. Break into sub-problems
3. Identify dependencies
4. Create task decomposition tree
5. Estimate complexity per sub-problem
