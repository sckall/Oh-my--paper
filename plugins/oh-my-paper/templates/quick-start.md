# Oh My Paper 快速开始指南

## 最简单的用法

```
# 1. 启动 Mega 自动模式（推荐）
/omp:mega
→ 选择 "自动执行"
→ 然后就不用管了！

# 2. 或者用 plan 来看状态和建议
/omp:plan
→ 它会告诉你下一步该做什么
```

## 自动执行的关键节点

| 阶段 | 自动执行什么 | 何时暂停 |
|------|------------|---------|
| A7 辩论 | `/omp:debate` 自动三轮辩论 | PIVOT 时暂停 |
| B14 分析 | `/omp:analyze` 独立 LLM 评估 | REFINE/PIVOT 时暂停 |
| A5/B10/C20 门控 | `/omp:gate` 自动检查 | 未通过时暂停 |

## 常用命令

| 命令 | 用途 | 何时用 |
|------|------|--------|
| `/omp:mega` | 25 阶段流水线 | 想全自动时 |
| `/omp:plan` | 查看状态和建议 | 不知道下一步时 |
| `/omp:ideate` | 生成创新点 | 开始新方向时 |
| `/omp:debate` | 辩论验证假设 | 假设生成后 |
| `/omp:experiment` | 实验循环 | 需要做实验时 |
| `/omp:analyze` | 独立 LLM 分析 | 实验结果出来后 |
| `/omp:write` | 论文写作 | 开始写论文时 |
| `/omp:review` | 同行评审 | 论文写完后 |
| `/omp:parallel` | 并行执行 | 想同时做多件事时 |

## 快速问答

**Q: 不知道该用什么命令？**
```
/omp:plan
它会告诉你该做什么
```

**Q: 想全自动跑完整个流程？**
```
/omp:mega
→ 选择 "自动执行"
```

**Q: 想看当前进度？**
```
/omp:progress
```

**Q: 想同步所有状态？**
```
/omp:sync
```

## 设置自动决策

如果想让流程更自动化，在 `.pipeline/memory/settings.md` 中设置：

```markdown
AUTO_EXECUTE: true     # 启用自动执行
AUTO_DECISION: false   # 决策仍需确认（推荐）
PAUSE_ON_PIVOT: true   # PIVOT 时暂停
```

---

## 干预方式

即使在自动执行中，你也可以随时说：

| 你的输入 | 效果 |
|--------|------|
| `暂停` | 停止自动执行 |
| `停止` | 完全停止 |
| `继续` | 从暂停处恢复 |
| `跳过` | 跳过当前阶段（谨慎）|
