# 写作风格指南：Computers & Education

本指南基于5篇已发表论文的分析生成，旨在帮助作者调整写作策略以符合Computers & Education期刊的要求。

## 1. 标题（Title）

### 长度要求
- **推荐长度**：8-12个单词（约40-60字符）
- **最小长度**：6个单词
- **最大长度**：15个单词

### 结构模式
基于分析，该期刊偏好以下标题结构（按推荐程度排序）：

#### 1. 冒号分隔结构（推荐 ⭐⭐⭐⭐⭐）
**格式**：`主标题: 副标题`
**示例**：
- "Enhancing student engagement through AI: A mixed-methods study"
- "Learning analytics in higher education: A systematic review"

**优点**：
- 清晰表达研究主题和方法
- 便于数据库检索
- 吸引读者注意力

#### 2. 疑问句结构（推荐 ⭐⭐⭐⭐）
**格式**：`What/How/Why + 研究问题?`
**示例**：
- "What makes a good online teacher? Student perceptions and expectations"

**优点**：
- 引发读者兴趣
- 直接点明研究问题

#### 3. 描述性结构（推荐 ⭐⭐⭐）
**格式**：`研究主题 + 研究方法`
**示例**：
- "The role of self-regulated learning in online STEM education"

### 常用关键词
- **技术类**：AI, chatbot, learning analytics, dashboard, feedback
- **教育类**：student engagement, learning outcomes, online learning, teaching quality
- **方法类**：mixed-methods, quasi-experimental, systematic review

### 标题写作检查清单
- [ ] 长度在8-12个单词之间？
- [ ] 清晰反映研究内容和方法？
- [ ] 包含关键技术词（AI, learning analytics等）？
- [ ] 避免使用缩写（除非是广泛认可的）？
- [ ] 标题结构符合期刊偏好（冒号分隔或疑问句）？

## 2. 摘要（Abstract）

### 长度要求
- **推荐长度**：150-250词
- **绝对限制**：不超过300词（含关键词）

### 结构模板
该期刊偏好结构化摘要，建议按以下顺序组织：

```
[背景/问题] + [目的] + [方法] + [结果] + [结论] + [实践意义]
```

### 写作示例
```
Background: [1-2句] 描述研究背景和问题重要性
Purpose: [1句] 明确研究目的或研究问题
Methods: [2-3句] 描述研究设计、参与者、数据收集和分析方法
Results: [2-3句] 报告主要发现（包含具体数值）
Conclusions: [1-2句] 总结研究贡献和实践意义
Keywords: [4-6个] 选择检索友好的关键词
```

### 常用句式
- **背景**：`Recent advances in [技术] have shown potential for [应用]...`
- **目的**：`This study aims to investigate...` / `The purpose of this research is to...`
- **方法**：`Using a [定量/定性/混合] approach, we...` / `A total of [N] participants...`
- **结果**：`The results indicate that...` / `Significant differences were found in...`
- **结论**：`These findings suggest that...` / `This study contributes to...`

### 摘要写作检查清单
- [ ] 长度在150-250词之间？
- [ ] 包含研究背景、目的、方法、结果、结论？
- [ ] 报告了具体数值（如效应量、百分比）？
- [ ] 突出了实践意义（practical implications）？
- [ ] 包含4-6个关键词？
- [ ] 未引用文献？
- [ ] 未使用缩写（除非是广泛认可的）？

## 3. 引言（Introduction）

### 长度要求
- **推荐长度**：2-3页（约1500-2000词）
- **最小长度**：1.5页
- **最大长度**：4页

### 结构设计
基于分析，该期刊的引言通常包含以下要素（按出现顺序）：

#### 3.1 研究背景（0.5-1页）
- 使用统计数据说明问题的重要性
- 引用近5年的高质量文献（特别是2020年后的COVID-19相关教育研究）
- 从宏观到微观：全球趋势 → 国家/地区现状 → 具体研究问题

**常用句式**：
```
The COVID-19 pandemic has accelerated the adoption of online learning...
Recent studies indicate that [具体数据]...
Despite the growing interest in [技术/方法], little is known about...
```

#### 3.2 问题陈述（0.5页）
- 明确指出研究空白（research gap）
- 说明为什么这个问题重要
- 批判性地回顾现有研究的局限性

**常用句式**：
```
While existing research has focused on [A], less attention has been paid to [B]...
Previous studies suffer from [局限性1, 局限性2]...
There is a lack of [理论/实证/方法] understanding of...
```

#### 3.3 研究目标与假设（0.5页）
- 列出2-3个具体的研究目标
- 如有假设，清晰陈述（H1, H2...）
- 解释为什么这些目标是重要的

**常用句式**：
```
The aims of this study are to: (1) ..., (2) ..., and (3)...
Based on [理论], we hypothesize that...
This research seeks to answer the following research questions: RQ1: ... RQ2: ...
```

#### 3.4 主要贡献（0.3-0.5页）
- 列出3-4个具体贡献
- 说明理论贡献和实践意义
- 避免夸大，基于证据陈述

**常用句式**：
```
This study contributes to the literature in three ways: First, ... Second, ... Third, ...
The findings provide practical implications for [教育实践者/政策制定者]...
```

### 引言写作检查清单
- [ ] 长度在2-3页之间？
- [ ] 引用了近5年的相关研究（特别是2020年后）？
- [ ] 明确指出了研究空白？
- [ ] 列出了具体的研究目标或研究问题？
- [ ] 陈述了主要贡献（3-4个）？
- [ ] 未过早披露具体结果？
- [ ] 语言客观、避免主观判断？

## 4. 文献综述（Literature Review / Theoretical Framework）

### 长度要求
- **推荐长度**：3-4页（约2000-2500词）
- **最小长度**：2页
- **最大长度**：5页

### 结构设计
**重要**：不要按作者罗列文献！应按主题组织。

#### 4.1 主题1：[主题名称]（1-1.5页）
- 综述该主题的关键研究
- 指出研究方法、主要发现、局限性
- 使用子标题（如有多个子主题）

#### 4.2 主题2：[主题名称]（1-1.5页）
- 同上

#### 4.3 主题3：[主题名称]（如有，1页）
- 同上

#### 4.4 研究空白（0.5页）
- 综合前述综述，明确指出现有研究的不足
- 说明本研究的切入点

### 常用理论框架
基于分析，该期刊论文常使用以下理论：
- **TPACK模型**（技术-教学-内容知识）
- **TAM模型**（技术接受模型）
- **SRL理论**（自我调节学习）
- **社区探究理论**（Community of Inquiry）
- **社会建构主义**（Social Constructivism）

### 文献综述写作检查清单
- [ ] 按主题组织，而非按作者罗列？
- [ ] 包含了批判性分析，而非简单描述？
- [ ] 引用了足够数量的文献（25-40篇）？
- [ ] 近5年文献占比≥40%？
- [ ] 明确了研究空白？
- [ ] 如有理论框架，清晰说明了理论依据？

## 5. 方法论（Methodology）

### 长度要求
- **推荐长度**：3-4页（约2000-2500词）
- **最小长度**：2页
- **最大长度**：5页

### 结构设计

#### 5.1 研究设计（0.5页）
- 明确说明研究类型（定量/定性/混合）
- 解释为什么选择这个设计
- 说明研究的时间跨度、地点、情境

**常用句式**：
```
This study employs a [quantitative/qualitative/mixed-methods] research design...
A [experimental/quasi-experimental/correlational] design was chosen because...
Data were collected over [时间] at [地点]...
```

#### 5.2 参与者（0.5-1页）
- 描述参与者特征（年龄、性别、教育背景等）
- 说明抽样方法（随机抽样/便利抽样/目的抽样等）
- 报告样本量及计算依据（如适用）
- **伦理审查**：说明是否获得伦理批准（IRB/IEC）

**常用表格**：
```markdown
Table 1. Participant demographics
| Variable | Category | N | % |
|----------|----------|---|---|
| Gender   | Male     | 45| 30%|
|          | Female   |105| 70%|
| Age      | 18-22    | 85| 56.7%|
|          | 23-30    | 65| 43.3%|
```

#### 5.3 数据收集工具（1页）
- **问卷**：提供信度证据（Cronbach's α），如适用，提供效度证据
- **访谈提纲**：说明访谈问题的设计依据
- **测试题**：说明评分标准和信度
- **技术工具**：说明技术平台的功能和验证

**常用句式**：
```
The questionnaire consisted of [N] items measured on a [5/7]point Likert scale...
Cronbach's alpha for each construct was above [0.7/0.8], indicating good reliability...
The interview protocol was validated by [专家评审/pilot test]...
```

#### 5.4 数据分析方法（0.5-1页）
- **定量**：明确说明统计方法（描述性统计、推断性统计、效应量等）
- **定性**：说明编码方法（主题分析/内容分析等）、编码者间信度
- **软件**：说明使用的统计软件（SPSS, R, NVivo等）

**常用句式**：
```
Descriptive statistics (mean, standard deviation) were calculated for all variables...
Inferential statistics included [t-test/ANOVA/regression]...
Effect sizes were reported as [Cohen's d/η²]...
The qualitative data were analyzed using [thematic analysis] following [Braun & Clarke's, 2006] framework...
Inter-coder reliability was calculated using [Cohen's κ], which was [0.85], indicating high agreement...
```

#### 5.5 研究质量（0.5页）
- **信度**：内部一致性、重测信度等
- **效度**：内容效度、构念效度、外部效度等
- **局限性**：说明研究设计的潜在局限性（如抽样偏差、共同方法偏差等）

### 方法论写作检查清单
- [ ] 研究设计清晰明确？
- [ ] 参与者描述详细（可复现）？
- [ ] 数据收集工具的信效度证据提供？
- [ ] 数据分析方法具体且恰当？
- [ ] 说明了伦理审查情况？
- [ ] 承认了研究局限性？
- [ ] 其他研究者可以完全复现？

## 6. 结果（Results）

### 长度要求
- **推荐长度**：2-3页（约1500-2000词）
- **最小长度**：1.5页
- **最大长度**：4页

### 结构设计

#### 6.1 描述性统计（0.5页）
- 报告均值、标准差、频率分布等
- 使用表格或图表展示

**常用表格**：
```markdown
Table 2. Descriptive statistics
| Variable | M | SD | Min | Max |
|----------|----|----|-----|-----|
| Engagement | 3.85 | 0.72 | 1.5 | 5.0 |
| Satisfaction | 4.12 | 0.68 | 2.0 | 5.0 |
```

#### 6.2 假设检验（1-1.5页）
- 报告推断性统计结果（t-test, ANOVA, regression等）
- **必须报告**：检验统计量、自由度、p值、效应量、置信区间
- 使用表格或图表辅助说明

**常用句式**：
```
An independent-samples t-test revealed a significant difference..., t([df]) = [值], p = [精确值], d = [效应量]...
A multiple regression analysis showed that [变量] significantly predicted [结果], β = [值], t([df]) = [值], p = [精确值]...
```

**注意**：
- 报告精确的p值（如p = .032），而非p < .05
- 报告效应量（Cohen's d, η², R²等）
- 报告置信区间（如95% CI [下限, 上限]）

#### 6.3 补充分析（如有，0.5页）
- 中介效应分析
- 调节效应分析
- 亚组分析

### 图表规范
- **分辨率**：至少300 dpi
- **格式**：TIFF或EPS（期刊偏好）
- **标注**：清晰、完整、符合APA格式
- **标题**：简洁、信息丰富
- **颜色**：考虑色盲读者（使用Pattern或高对比度颜色）

### 结果写作检查清单
- [ ] 未包含主观解释（留在Discussion）？
- [ ] 报告了精确的p值？
- [ ] 报告了效应量？
- [ ] 报告了置信区间（如适用）？
- [ ] 表格/图表清晰、标注完整？
- [ ] 使用了恰当的统计术语？
- [ ] 结果与研究问题/假设对应？

## 7. 讨论（Discussion）

### 长度要求
- **推荐长度**：3-5页（约2000-3000词）
- **最小长度**：2页
- **最大长度**：6页

### 结构设计

#### 7.1 主要发现总结（0.5-1页）
- 简明扼要地总结主要发现
- 与假设对应（支持/不支持）

**常用句式**：
```
The results partially supported H1, which predicted that [假设内容]...
Contrary to our expectation (H2), we found that...
```

#### 7.2 与文献对比（1-1.5页）
- 说明本研究结果与先前研究的一致性和差异性
- 解释造成差异的可能原因
- 深化理论理解

**常用句式**：
```
Our finding that [结果] is consistent with [作者, 年份], who also found...
However, our results contradict [作者, 年份]'s claim that...
A possible explanation for this discrepancy is...
```

#### 7.3 理论贡献（0.5-1页）
- 说明本研究对教育理论的补充或挑战
- 提出理论修正或扩展（如适用）

#### 7.4 实践意义（⚠️ 重点！）（1-1.5页）
**这是Computers & Education期刊特别强调的部分！**

- 对教育实践者的建议（教师、教学设计者、教育管理者）
- 对政策制定者的建议
- 具体、可操作的建议（而非泛泛而谈）

**常用句式**：
```
For practitioners, this study suggests that...
Educational institutions should consider...
Policymakers need to...
To enhance [具体技能/能力], we recommend...
```

#### 7.5 研究局限（0.5页）
- 抽样局限（样本代表性、抽样偏差）
- 测量局限（测量工具的不足）
- 设计局限（横截面数据、自我报告偏差等）
- 外部效度局限（推广性）

**注意**：不要过度自贬，客观陈述即可。

#### 7.6 未来研究建议（0.5页）
- 基于研究局限，提出未来研究方向
- 具体、可操作的方向（而非泛泛而谈）

**常用句式**：
```
Future studies should consider using [更严格的设计]...
Longitudinal research is needed to...
It would be valuable to replicate this study in [不同情境]...
```

### 讨论写作检查清单
- [ ] 解释了主要发现（而非简单重复结果）？
- [ ] 与文献进行了对比？
- [ ] 强调了实践意义（practical implications）？
- [ ] 客观陈述了研究局限？
- [ ] 提出了具体的未来研究建议？
- [ ] 未引入新的结果或数据？

## 10. 通用写作规范

### 语言风格
- **语态**：以被动语态为主（约35%），但主动语态也可接受
- **时态**：
  - 引言、文献综述：现在时
  - 方法论、结果：过去时
  - 讨论、结论：现在时
- **人称**：避免使用第一人称（"I"或"We"），除非在方法论文节描述作者行动
- **语气**：客观、谨慎、避免绝对化表述（如"prove", "definitely"）

### 句子结构
- **平均句长**：15-25词
- **段落长度**：3-5句
- **过渡词**：适当使用（However, Furthermore, In contrast, Therefore等）

### 常见错误
- ❌ 使用缩写而未在首次出现时定义
- ❌ 报告p < .05而非精确p值
- ❌ 未报告效应量
- ❌ 在Results章节进行主观解释
- ❌ 文献综述按作者罗列而非按主题组织
- ❌ 夸大研究发现（"prove"而非"suggest"）
