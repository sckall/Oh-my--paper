\section{Method}
ViewerLeaf is organized into four layers: a desktop shell, a compile and SyncTeX layer, an agent runtime, and a figure workspace.

\subsection{Desktop Shell}
The shell keeps the project tree, source editor, PDF preview, and bottom status dock visible together.

\subsection{Agent Runtime}
Agent profiles expose different academic skills depending on whether the user is outlining, drafting, polishing, de-AIing, or reviewing the paper.

\subsection{Figure Workspace}
The figure workspace allows users to draft a figure brief, refine it with a skill, trigger banana generation manually, and decide whether to insert the resulting figure snippet.
