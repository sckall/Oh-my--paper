# 工作日志 - 2026-05-02

## 实现 OMP 插件更新功能

### 新增功能
1. **`/omp:update` 命令** - 一键更新插件到最新版本
   - 支持 `--check-only` 参数（仅检查更新）
   - 支持 `--auto` 参数（自动更新，不询问确认）
   - 从 GitHub 自动下载最新代码并安装到插件缓存目录

2. **自动更新检查机制**
   - 在 SessionStart hook 中增加检查逻辑（每日最多一次）
   - 发现新版本时提示用户运行 `/omp:update`
   - 检查记录存储在 `.pipeline/.last-update-check` 文件

### 新增文件
- `skills/omp:update/SKILL.md` - 更新技能文档
- `plugins/oh-my-paper/commands/update.md` - `/omp:update` 命令实现
- `plugins/oh-my-paper/scripts/check-update.mjs` - 版本检查脚本
