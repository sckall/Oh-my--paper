## 彻底解决 AI 幻觉问题（write.md 修改）

### 背景
用户要求：写论文不允许有任何数据造假、不科学、不严谨的地方。
引用造假（fake citations）是 AI 写论文的主要问题。

### Commit 1: d8e81ef — 引用真实性强制规则
- 新增「🔴 引用真实性强制规则」章节（最高优先级，不可跳过）
- 写前强制检查 references.bib / literature_bank.md 是否存在
- 无真实参考文献时，严禁写任何 `\cite{}`，改用 `[CITATION NEEDED]` 占位
- 有参考文献时，只允许使用真实存在的 citation key，严禁编造
- 每节写完后强制提取所有 `\cite{}` key，用 AskUserQuestion 展示确认
- 重写「第三步」，将引用审查改为强制步骤（不可跳过）
- 新增 verify-citations.py 脚本调用（CrossRef/arXiv/SemanticScholar 三层验证）
- 验证不通过时严禁进入 /omp:review 或 /omp:export

### Commit 2: f602ba8 — 数据与结果真实性强制规则
- 新增「🟬 数据与结果真实性强制规则」章节
- 实验数据必须来自 experiment_ledger.md / result_summary.md，严禁捏造
- 严禁编造 p-value、effect size、百分比等任何数字
- 禁用绝对化表述（proves→suggests，demonstrates→indicates）
- 要求 Discussion 中必须讨论局限性（样本量、外部效度等）
- 每节写完后强制核验关键数值来源

## 增强期刊分析功能（omp:journal-analyze）

### 背景
用户反馈：需要更详细的期刊风格分析来辅助写作。具体要求：
1. 提取论文中的图表，分析其风格特征
2. 增加微观写作技巧、段落级指导
3. 从高分论文中提取"好句子"，按功能分类

### 新增的3个分析文件

#### 1. figure-table-style.yaml（图表风格分析）
- 图表类型偏好（条形图40%、折线图25%等）
- 图表规范（分辨率≥300 dpi、格式TIFF/EPS、字体8-12pt）
- 配色方案（推荐ColorBrewer、避免红绿搭配）
- 图表标题与标注规范（Figure/Table Caption格式）
- 图表内容规范（数字精度、统计符号、缩写）
- 常见问题与避坑指南
- 投稿前图表检查清单

#### 2. micro-writing-guide.md（微观写作技巧指南）
- 段落结构（主题句+支撑句+分析/过渡句）
- 各章节的段落写作技巧：
  - 引言：从宏观到微观、批判性回顾文献、研究目标具体化
  - 文献综述：按主题组织（非按作者）、批判性分析
  - 方法论：辩护式写作、参与者描述详细
  - 结果：先总体后具体、完整报告统计结果
  - 讨论：联系假设、解释机制、强调实践意义
- 平滑过渡技巧（段落间过渡、句子间过渡）
- 常见微观写作问题（句子过长、被动语态过度使用等）
- 提升写作质量的微观技巧（强动词、避免模糊词汇等）

#### 3. good-sentences.yaml（好句子库）
- 按功能分类的句型模板：
  - 指出研究空白（5个模板+示例+中文翻译）
  - 报告结果（描述性统计、t-test、ANOVA、回归分析、定性发现）
  - 讨论局限性（抽样局限、测量局限、设计局限、共同方法偏差）
  - 陈述贡献（理论贡献、方法论贡献、实践意义）
  - 过渡与连接（段落间过渡、句子间过渡）
  - 引用文献（支持论点、对比文献）
- 每个模板包含：template（句型模板）、example（英文示例）、translation（中文翻译）
- 使用方法：确定写作功能→选择合适句型→替换为自己的内容→检查引用格式

## 解决 guide.md 命令不显示的根本问题

### 问题根因
`~/.claude/plugins/installed_plugins.json` 记录了 OMP 插件的安装信息：
```json
"omp@oh-my-paper": {
  "installPath": "~/.claude/plugins/cache/oh-my-paper/omp/1.0.0",
  "version": "1.0.0"
}
```
即使删除了缓存目录，Claude Code 仍然认为已安装的是 v1.0.0 旧版，不会从项目 `plugins/oh-my-paper/` 读取新命令（如 `guide.md`）。

### 修复
- 清空 `~/.claude/plugins/installed_plugins.json` 中的旧安装记录
- 删除 `~/.claude/plugins/cache/oh-my-paper/` 缓存目录
- 清除 `~/.claude/skills/omp:*` 旧版技能（25个）
